# Chances 个人使用验收清单

版本：`1.0.0-local`

## 启动与数据

- [ ] 双击 `Start-Chances.cmd` 后浏览器打开 `http://127.0.0.1:8501`。
- [ ] 页面顶部显示“接口已连接”。
- [ ] 首页能看到自己的目标岗位、南京和薪资要求。
- [ ] 岗位池能看到原有岗位，重启后数据仍然存在。

## 岗位导入

- [ ] 在 Chrome 扩展管理页重新加载 `extension/`。
- [ ] 打开一个受支持的真实岗位详情页。
- [ ] 点击“截图并导入当前页面”。
- [ ] 扩展显示中文成功或明确的中文失败原因。
- [ ] 新岗位能在导入审查区或岗位池中找到。
- [ ] 岗位详情能查看原始正文或截图证据。

## 判断与准备

- [ ] 对一个岗位执行统一评分。
- [ ] 查看画像匹配、优势、风险和建议动作。
- [ ] 生成岗位准备包。
- [ ] 准备包包含简历角度、项目亮点、HR 问题、面试准备和风险回应。

## 推进与复盘

- [ ] 修改岗位状态和下一步动作。
- [ ] 新建一条投递或沟通事件。
- [ ] 新建一个跟进任务并将其完成。
- [ ] 周复盘能显示状态、事件、任务和停滞岗位变化。
- [ ] Excel 导出可以正常下载和打开。

## 备份与停止

- [ ] 双击 `Backup-Chances.cmd` 后在 `backups/` 看到新备份。
- [ ] 备份目录包含 `jobs.db`、`captures/` 和 `manifest.json`。
- [ ] 双击 `Stop-Chances.cmd` 后前后端都停止。
- [ ] 再次启动后原有岗位、画像和任务仍然存在。
