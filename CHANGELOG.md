# 变更日志

## 1.0.0-local - 2026-06-13

Chances 首个个人自用稳定版本。

### 已完成

- FastAPI、SQLite、Streamlit 与 Chrome 扩展的本地求职工作流。
- 岗位采集、导入审查、岗位池、分页搜索和批量状态更新。
- 个人画像、PDF 简历导入、岗位匹配、统一评分和评分历史。
- 岗位准备包、简历改写建议、投递事件、跟进任务和周复盘。
- 页面截图证据与确定性视觉摘要。
- 前端 API 实时连接状态和明确的错误恢复提示。
- 一键启动、停止和状态检查。
- SQLite 在线备份、截图备份、安全恢复和隔离恢复演练。
- 健康检查、故障排查和个人流程验收清单。
